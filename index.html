PROYECTOS

A DÓNDE VAMOS?


Desarrolladora Front-End y Líder del Equipo en Xavia IOT (2023 / Presente)

Tecnologías: React.js, Next.js, TypeScript, Material-UI, GitHub, Trello.

Me encargué del diseño y desarrollo front-end de la interfaz para el usuario en 'A Dónde Vamos', una innovadora plataforma web que simplifica la búsqueda de restaurantes cercanos. Implementamos estas tecnologías para garantizar una experiencia de usuario fluida y eficiente, colaborando en un equipo que atendió a tres roles clave: usuario, comercio y administrador, mediante metodologías ágiles.

Durante el desarrollo de la plataforma web, enfrentamos el desafío de integrar la API de Google para mejorar la experiencia del usuario mediante la visualización de mapas y direcciones. Este proceso fue un verdadero desafío, pero también una invaluable oportunidad de aprendizaje que nos permitió fortalecer nuestras habilidades y conocimientos técnicos.

Además, hemos comenzado el desarrollo de la aplicación móvil de 'A Dónde Vamos' utilizando React Native, Expo y TypeScript.
