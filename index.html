A Dónde Vamos?

Desarrolladora Front-End, Diseñadora y Líder del Equipo

Xavia IOT 2023 / Presente

Diseñar y desarrollar interfaces de usuario cautivadoras y funcionales, liderando equipos para crear experiencias web intuitivas. Especializada en tecnologías como React.Js, Next.Js, TypeScript, Material-Ui, GitGub y Trello. Dirigir el diseño y el desarrollo de la interfaz para el usuario "A Dónde Vamos", una innovadora plataforma web que simplifica la búsqueda de restaurantes cercanos al usuario. Implementar tecnologías de vanguardia, incluyendo React.Js y Next.Js, para asegurar una experiencia de usuario fluida y eficiente. Liderar el equipo de trabajo en la creación de una página web que atiende a tres roles clave: usuario, comercio y administrador, mediante metodologías ágiles.
